Socksgo

socksgo is the most complete, compatible, featured and extensible socks library for go.
And above other features it supports WASM builds and socks-over-websocket protocol extension.

Prerequisites

  • A socks5 server with socks-over-websocket extension support must be running
  • You can start compatible gost server with: just start-proxy or gost -L socks5+ws://localhost:1080
  • Example socks-over-webSocket URL: socks5+ws://localhost:1080
